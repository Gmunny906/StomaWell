Description: This project is called the StomaWell Care Assistant, this web page will be a website that works in tandem with a product that is being patented right now that will give you care assistance for using the StomaWell table, the StomaWell is a table that will make the cleaning of a Stoma much easier and cleaner for the user. Instead of having to clean the stoma in the shower or somewhere that is not supposed to be messy, the StomaWell table will help with this. And the Care Assitant is a website that the user of the StomaWell can go to for tips on sanitization, more information on the StomaWell, care instructions, and more.
